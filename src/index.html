Ons verhaal

  • Google assistent

    Met de opkomst van de virtuele assistent hebben ontwikkelaars nieuwe mogelijkheden om API’s met conversationele interfaces te verbinden. We hebben de Google Assistent met bol.com verbonden en er een blog over geschreven om mede software engineers te inspireren. Met deze assistent kunnen we op een natuurlijke manier technologie gebruiken. Bol.com, een online retailer in nederland, stimuleert derden om met hun platform te verbinden door open API’s.

    We hebben de Catalog API en Google’s DialogFlow gebruikt om Intents te maken die door Natural Language Processing (NLP) geïnterpreteerd kunnen worden en zo naar producten te zoeken. Voor de combinatie van DialogFlow en NLP bleek dat een platte Intent structuur met een rijke context de beste manier was om gesprekken te modelleren. Het toevoegen van een Natural Language Interface aan een API creëert interessante nieuwe scenario’s voor de interactie met machines door mensen. In het verleden biedt een API een uitgebreide reactie en filtered de gebruiker de output. Wanneer je met je stem communiceert met een API moet de reactie duidelijk en to-the-point zijn. Data kwaliteit is ontzettend belangrijk, maar precisie is het allerbelangrijkst voor een goede Natural Language experience.
