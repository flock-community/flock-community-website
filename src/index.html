Over ons

Wij zijn een community van software engineers

Flock. is een community waar software engineers vol passie samenwerken. We delen kennis, inspireren elkaar en maken samen mooie software.

Het allerbelangrijkste is dat jij kan doen wat je leuk vindt.

Ons verhaal

  • Google assistent

    Flock heeft ervaring met google assistant applicaties ontwikkelen (zie: vorige blog post)

    We observeren een test "gat" bij ontwikkelen van deze applicaties. Ze moeten manueel getest worden om te garanderen dat alle "conversation flows" door het systeem blijven werken. Naar aanleiding daarvan heeft Flock een test framework ontwikkelt om deze flows geautomatiseerd te testen.

    Cucumber is de aangewezen tooling voor dit probleem: het kan scenario's in natuurlijke taal vertalen naar stappen die tegen de applicatie getest kunnen worden. Elke stap representeert een input (Als ik zeg "...") of een output (Dan zegt de assistent "..."). Hierdoor zijn de "conversation flows" opgeschreven als een "executable specification"; het geeft weer wat de applicatie allemaal kan dmv voobeeld conversaties en test gelijkertijd dat die conversaties mogelijk zijn.

    Dit test framework is uitgerold bij Bol.com op black friday 2018. Er wordt verder aan door ontwikkeld om ook alle flows op de telefoon te kunnen testen. Hier kan de assistent naast tekst ook reageren met visuele middelen.
